#### 介绍
这是一个本地的可部署在web端的用户管理界面
技术：flask，request

#### 项目特色
这是根据用户的身份，登录是同一个界面，登入后针对不同的页面返回不同的页面
采用授权等级的方式，自动判断登陆者的身份

#### 安装方法
下载文件后，配置环境
pip install flask,request

#### 使用方法
运行main.py，点击跳转的网页即可
